Url.dll简介：

url.dll是Internet快捷壳扩展相关应用程序接口系统文件。

说明：url.dll所在路径已被系统添加PATH环境变量中，因此，url.dll命令可识别，但由于为dll文件，需调用rundll32.exe来执行。

Windows 2003 默认位置：

C:\Windows\System32\url.dll
C:\Windows\SysWOW64\url.dll

Windows 7 默认位置：

C:\Windows\System32\url.dll
C:\Windows\SysWOW64\url.dll

靶机： 192.168.1.3 Windows 7
